# Personal Portfolio
**Sleek dark-mode portfolio** built with React showcasing UI/UX, frontend work, and projects.  
Features a typewriter hero that cycles through **UI/UX Designer**, **MERN Stack Developer**, and **Programmer**, an animated avatar orbit, projects with Live Demo / View Details buttons, skills, and a contact form.

## Key Features

- Dark, professional UI with premium visual styling
- Hero section with typewriter effect (types and deletes each role)
- Animated avatar orbit with floating UI/UX tags
- Projects grid with **View Details** and **Live Demo** buttons
- Core Skills section with animated progress bars
- Contact form that opens user email client (`mailto:`)
- Sticky header with desktop navigation and a mobile hamburger menu
- Responsive layout for desktop and mobile

---

## Tech Stack

- **React** (Create React App or Vite)  
- **HTML & CSS** (custom modern styling, animations)  
- No backend required (contact form uses `mailto:`)
